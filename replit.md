# angejr - Next.js Project

## Overview
A Next.js 16 application with React 19, TypeScript, and Tailwind CSS.

## Project Structure
- `app/` - Next.js App Router pages and layouts
- `public/` - Static assets

## Development
- Run: `npm run dev` on port 5000
- Build: `npm run build`
- Start production: `npm start`

## Configuration
- Next.js config allows Replit proxy origins
- Uses Tailwind CSS 4 with PostCSS
